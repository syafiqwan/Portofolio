# Portfolio Website

## Deskripsi

Website Portfolio Pribadi yang dibuat menggunakan HTML dan CSS dasar. Website ini berisi informasi tentang profil, riwayat pendidikan, pengalaman, keahlian, proyek, dan galeri kegiatan.

## Fitur

* Halaman Home
* Halaman About
* Halaman Project
* Halaman Gallery
* Navigasi antar halaman
* Responsive Layout sederhana
* Link media sosial dan email

## Teknologi yang Digunakan

* HTML5
* CSS3

## Struktur Project

portfolio/

├── index.html

├── about.html

├── project.html

├── gallery.html

├── style.css

├── profil.jpeg

└── images/

    ├── gallery1.jpg

    ├── gallery2.jpg

    ├── project1.png

    └── dst

## Cara Menjalankan

1. Download atau clone repository.
2. Buka folder project.
3. Jalankan file `index.html` menggunakan browser.
